1. Install Postman web app
2. Install Newman
- This is nodejs app and needs nodejs, please get latest stable from here: https://nodejs.org/en/download/
- After nodejs is installed, open a console and run:
npm install -g newman
- To verify newman isntallation:
newman -h
3. Show portman GUI
- Send requet
- Add verifications
- Show variables
- Console.log() to debug -> chrome://inspect/#apps -> select postman
4. Export tests
5. Run postman via commandline
- Examples:
newman -c Lecture-04-Homework-Everlive-JustDemo
newman -u https://www.getpostman.com/collections/da7f69fac69caa22dc63

What to know before I start learning:
- VIM is different from everything else
- Don't start using it in production at the first day (first learn how to copy / paste)
- You will stop using your mouse - you don’t need the mouse
- Can i do ....? - Yes you can like your old text editor but better
- There is always a better way to doing it
- Try to learn something every week
